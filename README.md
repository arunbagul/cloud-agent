# cloud-agent

Cloud Agent is Cloud monitoring agent for Linux.

For older release please check http://openlsm.sourceforge.net.

* How to install cloud-agent

Ubuntu/Debian-
  # apt-get install curl libcurl3 libconfig9

  # apt-get install  cldmon
  
Redhat/CentOS

  # yum install cldmon
